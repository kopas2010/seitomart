# 政党マート

## サイト概要
### サイトテーマ
選挙情報を調べたり、投票先を選んだりするのに利用するためのサイト
候補者の情報や経歴、施策などをまとめた候補者ページなどを作成予定

### テーマを選んだ理由
- 候補者の情報を調べる機会がない
- 政治離れ防止

### ターゲットユーザ
- 投票先に悩んでる人

### 主な利用シーン
- 選挙前
